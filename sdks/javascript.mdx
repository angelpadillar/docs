## ðŸš€ Quick Start

```typescript
import { SDK } from "@buena/sdk";

// Initialize the client
const buena = new SDK({
  apiKey: "your-api-key",
  serverUrl: "https://api.buena.ai/api/v2",
});

// List leads
const leads = await buena.leads.list();
console.log(leads);

// Create a new lead
const newLead = await buena.leads.create({
  firstName: "John",
  lastName: "Doe",
  email: "john@example.com",
  company: "Example Corp",
});
```

## ðŸ“‹ Available Operations

### Leads Management

- `buena.leads.list()` - List all leads
- `buena.leads.create()` - Create a new lead
- `buena.leads.update()` - Update a lead
- `buena.leads.delete()` - Delete a lead

### API Keys Management

- `buena.apiKeys.list()` - List API keys
- `buena.apiKeys.create()` - Create API key

### Health Check

- `buena.health.check()` - Check API status
